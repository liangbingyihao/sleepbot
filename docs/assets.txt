用户早睡鼓励素材模块说明

1. 素材的产生：
1.1 用户在app客户端调用后端接口，生成一个上传素材的session，保存在数据库里,该事务有有效时长。返回包含新事务id（uuid）的一个网页链接给前端。
1.2 用户把该链接发给他的好朋友，好朋友在手机,pc等打开该素材采集页面。
1.3 如果在事务有效期内，允许好友上传图片，录制音频，或定义一段话。如果在有效期外，那则不能打开，需要用户再次生成新的事务。
1.4 后端接收到好友上传的图片或音频时，根据用户的区域（region）信息，拿到阿里云合适的Bucket ，通过 alibabacloud_oss_v2 接口保存到oss对象存储。并把保存的结果保存到数据库
数据库文件表类似这样 user_oss_files（不变，每条文件一行）
sql
user_id
friend_name 
bucket       -- 存：cn-bucket 或 sg-bucket
object_key   -- 如：user123/xxx.mp3
file_size
mime_type
created_at

1.5 上面说的"好友"不必是平台的用户。素材采集的界面要求好友输入他的昵称，昵称+用户id是系统内唯一约束的。好友在系统内记录下来的文本或上传的文件，同时也记录下好友的昵称。

1.6 合理规划object_key，方便oos控制台或其他模块管理。

2. 素材审核使用
2.1 用户登录后，调用接口，获取到收到的鼓励素材和好友的信息。
2.2 用户审核并决定采用或丢弃这些素材。后续用户在app里锁屏时，可以看到这些审核通过的素材。
2.3 当素材是图片或者音频时，需要调用遍历user_oss_files ，再调用alibabacloud_oss_v2获取预签名URL，有效期15分钟。
   pre_result = client.presign(
        oss.GetObjectRequest(
            bucket=args.bucket,  # 指定存储空间名称
            key=args.key,        # 指定对象键名
        )
    )


要求你开发上面涉及到的素材采集前端页面，和后端系列接口
